# FOSS Estate

An open source project to bring real estate management to everyone without breaking the bank

# The concept

FOSS Estate is a project that will in the long run allow real estate managers (landlords, housing agencies, etc...) to run their systems without having to rely on corporations that have the possibility of compromising their workflow (voluntarily or not).

## The structure

FOSS Estate will follow a star architecture:

- a self hostable web server with a database and accessible through a dashboard (/php)
- a website (/www)
- a web app (/src)
- a mobile app (/app)

# The technologies

The technologies, namely the languages and frameworks/libraries will be listed per section:

- Server : PHP, MySQL, Symfony
- Website : React (HTML, CSS, JS/JSX), Tailwind
- Web App : Java, Swing
- Mobile App: Kotlin
